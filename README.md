# tema_4_java

En este proyecto creamos una clase SmartDevice que otorga sus atributos a las clases hijas SmartPhone y SmartWatch.
Se generan 2 constructores, uno vacio y uno que contiene todos los parametros de las clases.
En el main se generan 2 objetos, un objeto de la clase Smartphone y otro de la clase SmartWatch.
Los parametros de cada objeto son establecidos e impresos por consola
